# Arduino-LED-Tetris
Fully playable Tetris game built on an Arduino Uno. Features a 4x8x8 LED matrix display, analog joystick controls, LCD score screen, and a passive buzzer for sound effects. Includes all 7 tetriminos, wall kicks, soft drop, next piece preview, line clear animations, high score saved to EEPROM, and a Fisher-Yates shuffle startup animation.
